# koodo-reader-plugins
My Koodo Reader plugins. Local proxy is required.

See [koodo-reader/plugins](https://github.com/koodo-reader/plugins) for more details.

## Usage

**Start Proxy Server**

```bash
npm install
node proxy.js
```
